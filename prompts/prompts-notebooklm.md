# Prompts Utilizados no NotebookLM

## Prompt de resumo
"Resuma este conteúdo em linguagem simples, destacando os conceitos principais
e utilizando exemplos do cotidiano."

## Prompt de pensamento crítico
"Quais decisões financeiras equivocadas este conteúdo ajuda a evitar?"

## Prompt de comparação
"Compare este material com as outras fontes do notebook,
apontando semelhanças e diferenças."

## Prompt de ensino
"Explique este conteúdo como se fosse para um estudante iniciante,
utilizando metáforas e exemplos simples."
